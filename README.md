# DevOps Practice Lab

This repository is my hands-on playground for exploring **DevOps tools** and workflows.  
It contains practical setups, sample configurations, and small projects covering:

## 🛠 Tools & Technologies
- **Docker** – Containerization & image management
- **Kubernetes** – Orchestration & scaling
- **Ansible** – Automation & configuration management
- **Terraform** – Infrastructure as Code (IaC)
- **Jenkins** – CI/CD pipeline automation

## 📂 Structure
- `/docker/` – Dockerfiles, Compose setups
- `/k8s/` – Kubernetes manifests
- `/ansible/` – Playbooks & inventories
- `/terraform/` – Infrastructure definitions
- `/jenkins/` – Pipeline scripts & configs

## 🚀 Goals
- Practice deploying applications in various environments
- Learn infrastructure automation
- Implement CI/CD workflows
- Experiment with scaling & monitoring
